# Как запустить сервер проверки иероглифов

## 1. Установить зависимости

Открой терминал в папке `server` и выполни:

```
cd server
npm install
```

## 2. Настроить Gmail (один раз)

Нужен пароль приложения для отправки писем:

1. Включи **2-этапную аутентификацию** в Google: https://myaccount.google.com/security
2. Создай **пароль приложения**: https://myaccount.google.com/apppasswords
3. Выбери «Другое» → придумай название → скопируй пароль (16 букв без пробелов)

## 3. Запустить сервер

В папке `server` выполни (замени `ТВОЙ_ПАРОЛЬ` на скопированный пароль):

```
set EMAIL_PASS=ТВОЙ_ПАРОЛЬ
node server.js
```

Сервер запустится на `http://localhost:3000`.

## 4. Сделать сервер доступным в интернете

Чтобы кнопки в письме работали, серверу нужен публичный URL.

**Вариант A — ngrok (проще всего для теста):**
1. Скачай ngrok: https://ngrok.com/download
2. Запусти: `ngrok http 3000`
3. Скопируй URL вида `https://abc123.ngrok.io`

**Вариант B — Render (бесплатный хостинг):**
1. Заливай папку `server` на GitHub
2. На Render.com → New Web Service → подключи репозиторий
3. Start Command: `node server.js`
4. Добавь переменную окружения `EMAIL_PASS`

## 5. Обновить URL в приложении

В файле `js/app.js` найди строку:

```js
const REVIEW_SERVER = 'http://localhost:3000';
```

Замени на твой публичный URL (например `https://abc123.ngrok.io` или `https://chinese-review.onrender.com`).
